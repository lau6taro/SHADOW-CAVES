# Caves of Shadows — Landing Page

## Descripción
Landing page interactiva de un videojuego tipo tower defense medieval.
Incluye personalización del usuario e interaccion dinámica con JavaScript.

## Tecnologías
- HTML
- CSS
- JavaScript (DOM)

## Funcionalidades
- Cambio de tema (oscuro/claro)
- Personalización del protagonista
- Trailer interactivo
- Easter egg secreto
- Galería de personajes dinámica

## Cómo ejecutar
1. Descargar o clonar el repositorio
2. Abrir index.html en el navegador
